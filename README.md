# BetterMorphing+

BetterMorphing+ is a gameplay-focused Hytale server mod that expands the morph potion system with new forms, balance tweaks, and custom behaviour for morphed players.

The goal is to make morphing feel powerful and fun, but still fair in survival gameplay.

## Features

- New morph potions based on existing Hytale creatures
- Pigeon morph with true flight and fall-damage immunity while morphed
- Hawk morph sharing pigeon-style flight, with a limited duration morph effect
- Frog morph with enhanced underwater breathing and mobility
- Dog and mouse morphs with significant movement speed bonuses
- Global balance pass on morph combat:
  - Reduced outgoing damage while morphed
  - Increased incoming damage while morphed
  - Taking damage can interrupt flight-based morphs and drop the player
- Integrated asset pack:
  - Custom item JSON for morph potions and recipes
  - Entity effect assets for morph states
  - Icons for morph potions
